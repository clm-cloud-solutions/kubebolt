# KubeBolt 1.5.0 — The AI Copilot release

KubeBolt 1.5.0 is the most complete open-source version we've shipped.
It turns the Copilot from a nice-to-have chat into a production-grade
troubleshooting partner, adds real admin analytics for AI usage, a
multi-channel notification stack, and 100 tests across backend and
frontend so you can trust what you deploy.

No vendor lock-in, no phone-home, no credit system. Bring your own
Kubernetes, bring your own AI key, run it anywhere.

## Highlights

### An AI Copilot that actually knows its place

The Copilot now understands two things it didn't before: **your
conversation** and **itself**.

- **Conversation memory.** Long debugging sessions no longer explode
  the context window. When the conversation approaches 80% of the
  configured budget, KubeBolt folds older turns into a summary using
  the cheap-tier model of your provider (Haiku 4.5 / gpt-4o-mini).
  You stay in the same session; the counter drops; the Copilot keeps
  going. One click on the scissors icon does the same thing on
  demand — useful when you're switching topics.
- **Product knowledge base.** The Copilot can now answer "How do I
  port-forward to a pod from the UI?" or "What does the scissors
  button do?" without hallucinating. 25 curated topics, on-demand, via
  a dedicated tool call.
- **Scope guardrail.** Ask the Copilot for a recipe or a poem and it
  will refuse, politely, in your language — then redirect you to what
  it can actually help with. No partial answers, no off-topic drift.
- **Contextual "Ask Copilot" buttons** now live on Insight cards,
  Resource Detail pages (Pods, Deployments, StatefulSets, Services,
  Nodes), and every Warning event in the Events page. One click
  launches the Copilot with the relevant context pre-loaded —
  cluster, namespace, resource name, symptom.
- **Multi-LLM, multi-provider, smart about tokens.** Anthropic and
  OpenAI with automatic prompt caching (up to 90% discount on cached
  reads), GPT-5 / o-series support via `max_completion_tokens`, and a
  fallback provider that kicks in on rate limits or 5xx.

### Admin analytics that answer "What's my Copilot costing me?"

New `/admin/copilot-usage` page. Persistent storage in the existing
BoltDB. 30-day retention. Configurable 24h / 7d / 30d range.

- Sessions, tokens billed, cache hit rate, estimated USD cost.
- Timeseries chart: cache reads vs fresh input vs output, stacked.
- Top tools with call counts and error rates.
- Recent sessions table with a drill-down modal showing tool
  breakdown and every compaction event.

Because you run it, the data stays with you. No telemetry, no SaaS
account required to see your own spend.

### Notifications you can actually tune

Slack, Discord and email channels with a global master toggle for
maintenance windows, a base URL for deep links back to KubeBolt, and
resolved-insight alerts so you know when a fire goes out. Email
supports instant, hourly and daily digest modes to keep noise down.

### A test suite to back everything up

86 Go tests + 14 frontend tests, run on every push and PR with the
race detector on. Catches real bugs (this release fixed a silent
usage-store pruning bug the tests surfaced on day one) and gives you
confidence to upgrade.

### Distribution polish

Still a single Go binary with the frontend embedded. Already available
via Helm OCI, Docker multi-arch, Homebrew tap, krew plugin, raw
binaries for Linux / macOS / Windows (amd64 + arm64). All release
artifacts signed with cosign.

## Upgrade

### Helm

```bash
helm upgrade kubebolt oci://ghcr.io/clm-cloud-solutions/kubebolt/helm/kubebolt --version 1.5.0
```

### Docker Compose

```bash
cd deploy && docker compose pull && docker compose up -d
```

### Homebrew

```bash
brew upgrade clm-cloud-solutions/tap/kubebolt
```

### krew

```bash
kubectl krew upgrade kubebolt
```

## Configuration

New environment variables introduced in 1.5.0. All optional — sensible
defaults everywhere:

```bash
# Conversation memory
KUBEBOLT_AI_AUTO_COMPACT=true
KUBEBOLT_AI_SESSION_BUDGET_TOKENS=         # 0 = auto from model
KUBEBOLT_AI_AUTO_COMPACT_THRESHOLD=0.80
KUBEBOLT_AI_COMPACT_MODEL=                 # empty = auto cheap tier
KUBEBOLT_AI_COMPACT_PRESERVE_TURNS=3

# Notifications — global
KUBEBOLT_NOTIFICATIONS_ENABLED=true
KUBEBOLT_NOTIFICATIONS_BASE_URL=
KUBEBOLT_NOTIFICATIONS_INCLUDE_RESOLVED=false

# Email (new)
KUBEBOLT_SMTP_HOST=
KUBEBOLT_SMTP_PORT=587
KUBEBOLT_SMTP_FROM=
KUBEBOLT_SMTP_TO=                          # comma-separated
KUBEBOLT_SMTP_DIGEST_MODE=instant          # instant | hourly | daily
```

Full reference: [`.env.example`](https://github.com/clm-cloud-solutions/kubebolt/blob/v1.5.0/.env.example).
